POST

3. 로그아웃 하기 
DELETE 
http://localhost:3000/friends/sign_out.json?email=iam1494@gmail.com

4. Auth test
GET 
http://localhost:3000/users/1.json?auth_token=6zPja98mDxG9bX7xSdqc

5. Product 가져오기
